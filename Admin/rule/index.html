ROY Parkour運営規約

<本規約について>
本規約はroyparkourの運営内で、定めるものです。
運営申請を送信した時点で本規約を熟読し、内容に同意したと見なしますのでしっかりご確認ください。
本規約は運営申請審査を通過した後に適応されます。
内容は変更される可能性があります。その際は日時を予告して運営にご連絡をするので必ずご確認ください。
規約変更の予告は適用予定の2日以上前にしなくてはならない。
規約変更適用前に内容について反対意見の連絡がなければ、変更後の規約内容に同意したものとみなす。

<定義>
「運営」「op」とはROY Parkourにて権限を保持し、鯖発展のため、ユーザーにコンテンツを提供するボランティアを指す。
「鯖主」とはminecraftID「ROOYROOY」を表す。ROY Parkourの代表。X(@ROOYROOY10)
「ユーザー」とはROYPARKOUR に参加可能なプレイヤー全員を指す。


<運営者の禁止事項>
一:荒らし、妨害行為
二:他ユーザーへの誹謗中傷、名誉毀損
三:ユーザーのプレイを妨害あるいは手助けする行為
四:不正行為(デバッグの場合を除く)でのアスレクリア(もし、過失で行ってしまった場合、しっかりと鯖主に報告してください。)
五:鯖主に無断で権限をユーザーに譲渡する行為
六:予告・あるいは許可なしにサーバーに異常な負荷をかける行為
七:機密情報(運営専用discordの会議内容も含む)を許可なく公開および伝達する行為
八:サーバールールの禁止事項に抵触する行為
九:他ユーザーの製作物を許可なく改変、複製、配布をする行為

<解雇について>
上記の「運営内の禁止事項」に抵触する行為を行った場合は解雇する可能性がある。

<解雇決定について>
鯖主の独断で決める(緊急事態の場合)あるいは
運営の一員から解雇提案があった場合、投票で決めます。OPの５割以上(小数になった場合は四捨五入します。)が投票した場合、解雇投票は成立し、対象者を除く投票者の8割以上(小数点以下四捨五入)が解雇を希望した場合、解雇が決定されます。

解雇が決まった場合はメンションもしくはDMでご報告いたします。
解雇された場合、情報漏洩防止のため、運営専用discordサーバーから追放させていただきます。


<運営申請条件>
1:13歳以上であり、未成年者であった場合、親に了承を得ていること。
2:Discordまたはtwitterアカウントを所持しており、かつ連絡が取れる状態であること。
3:上記の禁止事項を守れること。
4:ROY Parkourの運営に必要な一定以上の技術があること。
5:運営としてふさわしい人であること。
6:Minecraft Java Editionを所持していること。
注意事項
突破が不可能なアスレは作らないこと。必ずそのギミックできるか確認を！
プレイヤーにTPしたいときはスペクテイターモードにしてからTPしてください。(/tp3 <player>で自動でその処理を行ってくれます。)
