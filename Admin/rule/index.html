ROY Parkour運営規約

<本規約について>
本規約はroyparkourの運営内で、定めるものです。
運営申請をした時点で本規約を熟読し、内容に同意したと見なすのでしっかりご確認ください。
本規約は運営申請が通った際に適応されます。
また、内容は許可なく突然変更される可能性があります。その際は連絡をするので必ずご確認ください。
規約内容変更直後、鯖主に規約内容について反対意見の連絡がなければ、運営は変更後の規約内容に同意したものとします。

<定義>
「運営」「op」とはROY Parkourにて権限を保持し、鯖発展のため、ユーザーにコンテンツを提供するボランティアを指す。
「鯖主」とはminecraftID「ROOYROOY」を表す。また、ROY Parkourの代表を指す。
「ユーザー」とはROYPARKOUR に参加している人を指す。
「やる気のない」はサーバーに参加する回数が異常に少ない、サーバーに参加してても作業を全くしないことを指す。

<運営内の禁止事項>
第一条(以下一部省略):荒らし、妨害行為
2:他ユーザーへの誹謗中傷、名誉毀損
3:ユーザーのプレイを妨害する行為
4:不正行為でのアスレクリアおよびTAGの取得(もし、過失で行ってしまった場合、しっかりと鯖主に報告してください。)
5:鯖主に無断で権限を譲渡する行為
6:運営としてふさわしくない行動および発言
7:サーバーに異常な負荷をかける行為
8:機密情報(運営専用discordも含む)内容を許可なく公開および伝達する行為
9:サーバールールの禁止事項に抵触する行為
10:他ユーザーの製作物を許可なく改変、複製、配布をする行為

<解雇について>
やる気がないと感じられる、或いは上記の「運営内の禁止事項」に抵触する行為を行った場合は解雇を行う場合がある。
しばらくサーバーに来れない事情などがあれば必ずご連絡ください！
 (もし、連絡がなかった場合は、解雇対象になる可能性があります。)
[連絡があった場合、サーバーへの不参加という理由での解雇は一定期間行いません。]
 また、鯖運営陣として厳しい技術であると運営の8.5割以上が判断した場合、解雇も検討します。

<解雇決定について>
主の独断で決める(よほどのことがない限り行わないと思っていただいて結構です)あるいは
投票で決めます。OPの５割以上(小数になった場合は四捨五入します。)が投票した状態で、投票者の8割以上(対象者は除く。小数になった場合は四捨五入します。)が解雇を希望した場合、解雇が決定されます。

解雇が決まった場合はメンションもしくはDMでご報告いたします。
解雇されたかたは情報漏洩防止のため、運営専用discordからKick　させていただきます。


<運営申請条件>
1:13歳以上であり、未成年者であった場合、親に了承を得ていること。
2:Discordまたはtwitterアカウントを所持しており、かつ連絡が取れること。
3:上記の禁止事項を守れること。
4:ROY Parkourに必要な一定以上の技術があること。
5:運営としてふさわしい人であること。
6:Minecraft Java Editionを所持していること。
注意事項
突破が不可能なアスレは作らないこと。必ずそのギミックできるか確認を！
プレイヤーにTPしたいときはスペクテイターモードにしてからTPしてください。(/tp3 <player>で自動でその処理を行ってくれます。)
